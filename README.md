# Cowworker

Веб-сервис для поиска и бронирования коворкингов.

## Описание

Cowworker — учебный проект, разработанный в рамках итоговой аттестационной работы по программе «Фронтенд и бэкенд разработка».

Приложение позволяет:
- просматривать каталог коворкингов;
- выполнять поиск и фильтрацию;
- оформлять бронирование;
- просматривать список созданных бронирований;
- проверять API через Swagger.

## Используемые технологии

- HTML5
- CSS3
- JavaScript
- ASP.NET Core Minimal API
- JSON
- Swagger / OpenAPI
